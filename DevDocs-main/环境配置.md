## Unity
 **Unity Hub 和 Unity Editor 版本不能有类似 c1 、c2 、c3 等后缀。**

目前使用的 Unity Hub 版本为 3.5.1，Unity Editor 版本为 2022.3.62f1。

可在 [公司 FTP 站点](http://code.zhongbosoft.com/chrischen/Tutorials/-/blob/master/README.md#user-content-ps-%E5%87%A0%E4%B8%AA%E5%86%85%E7%BD%91%E5%85%AC%E5%85%B1%E8%B5%84%E6%BA%90) 下载安装，如自行下载安装，需要在 [unity.com](https://unity.com/) 下载。由于中国的 IP（包括香港）在下载 Unity 时会跳转到 [unity.cn](https://unity.cn/) ，此网站只提供 2022.3.62f1c1 下载，所以请使用其他 IP（例如美国、日本等）。

Unity Hub 与 Unity Editor 破解参考 [Uni-HubHacker](https://github.com/parhelia512/Uni-HubHacker)
